# Important Commands

---

## Homes

Set and teleport to personal saved locations.

| Command | What It Does |
|---------|--------------|
| `/sethome <name>` | Save your current location as a home |
| `/home <name>` | Teleport to a saved home |
| `/delhome <name>` | Delete a saved home |
| `/homes` | List all your saved homes |

---

## Warps

Server-wide teleport points set by staff.

| Command | What It Does |
|---------|--------------|
| `/warp <name>` | Teleport to a warp point |
| `/warps` | List all available warps |
| `/hub` | Return to the Hub world |

Common warps: `survival`, `smpworld`, `creative`, `oldhub`, `hide_and_seek`

---

## Claims

See the [Claims](/claims/) page for the full guide.

| Command | What It Does |
|---------|--------------|
| `/claimslist` | Show your claims and available blocks |
| `/trust <player>` | Give a player full access to your claim |
| `/untrust <player>` | Remove a player's access |
| `/abandonclaim` | Delete the claim you are standing in |

---

## General

| Command | What It Does |
|---------|--------------|
| `/help` | Show available commands |
| `/spawn` | Teleport to world spawn |
| `/tpa <player>` | Request to teleport to a player |
| `/tpaccept` | Accept a teleport request |
| `/tpdeny` | Deny a teleport request |
| `/msg <player> <message>` | Send a private message |
| `/r <message>` | Reply to the last private message |
